#

This demonstrates the use of BlankLink.

```jsx
const React = require('react');
const BlankLink = require('./BlankLink').default;

const BlankLinkExample = () => (
  <div>
    <BlankLink href="https://geops.de">
      Visit the geOps website!
    </BlankLink>
  </div>
);

<BlankLinkExample />;
```
